# 📰 游戏圈日报 - 数据目录

## 上传规则

### 文件命名
```
YYYY-MM-DD.txt
例如：2026-05-11.txt
```

### 文件格式（与原格式完全一致）

```
2026-05-11游戏圈日报，共 10 篇精选文章

1. ⭐ 重点推荐 文章标题
公众号：公众号名称   分类：行业动态
摘要：文章摘要内容
发布时间：2026/05/11 09:00
原文链接：https://mp.weixin.qq.com/s/xxxxxx

2. 文章标题
公众号：公众号名称   分类：出海
摘要：文章摘要内容
发布时间：2026/05/11 10:00
原文链接：https://mp.weixin.qq.com/s/xxxxxx
```

### 分类可选值
- `行业动态`
- `出海`
- `AI技术`
- `小游戏`

### 注意事项
- 每行末尾不要有多余空格
- `⭐ 重点推荐` 标记加在标题前，会高亮显示
- 文件编码使用 **UTF-8**
